GitHub = hosting para repos Git + colaboración (issues, pull requests, wikis).

Control de versiones: Git registra cambios de tus archivos mediante commits; GitHub aloja los repos en la nube.

Colaboración: ramas y pull requests permiten revisar, comentar y fusionar cambios de forma segura.

Portafolio: los repos públicos muestran tu trabajo y tu historial de contribuciones.

Piezas clave
  • Repositorio: código y archivos versionados
  • Pull Request: propuesta para integrar una rama
  • Issues: tareas, bugs y mejoras
